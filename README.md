# 📊 Universal Data Analysis & Machine Learning Platform

A comprehensive and versatile web application built with **Python**, **Streamlit**, and **Scikit-Learn** that allows users to upload any tabular dataset (CSV, Excel, JSON, Parquet, TXT, TSV) to perform end-to-end data cleaning, exploratory data analysis (EDA), advanced visualizations, predictive modeling, and AI-powered risk mitigation.

## 🛠️ Key Features
* **Universal File Loader:** Auto-detects delimiters (commas, semicolons, tabs) and supports multiple formats.
* **Interactive Data Cleaning & Schema Manager:** Dynamically recommends columns to drop (high-null or IDs) and lets users override variable types (Numerical vs. Categorical).
* **Exploratory Data Analysis (EDA):** Performs automated statistical profiling.
* **Advanced Visualizations:** Renders interactive Heatmaps, Scatter, Box, Violin, and Line plots with automated analytical insights and outliers detection.
* **Machine Learning Pipeline:** Trains both **Classification** and **Regression** models (Random Forest) with adjustable hyperparameters, performance metrics, and a real-time prediction simulator.
* **Excel Equivalents & KPI Strategies:** Displays exact Excel formulas for data cleaning based on the loaded dataset, and suggests business solutions.
* **Risk & Financial Mitigation (Fase 6):** Detects 3-Sigma budget anomalies, trains preventive risk models, and plots risk probability curves and feature importances.
* **AI-Powered Chatbot:** Integrated Google Gemini assistant to converse directly with your dataset.

## 📦 Tech Stack
* **Frontend/Deployment:** Streamlit
* **Data Processing:** Pandas, NumPy, OpenPyXL, PyArrow
* **Visualization:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn
* **Artificial Intelligence:** Google Generative AI (Gemini API)
